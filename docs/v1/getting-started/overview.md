# Overview

OR3 Intern is a personal AI agent. One binary, many ways to use it.

## What you can do

**Chat in your terminal**

```
or3-intern chat
```

Talk to your agent like you would a coworker. Ask it to run commands, check files, or research topics.

**Run it as a web service**

```
or3-intern service
```

This starts an HTTP API on port 9100. Connect the OR3 App on your phone or tablet. Or use any HTTP client.

**Connect messaging apps**

Connect Telegram, Slack, Discord, WhatsApp, or Email. Talk to your agent from wherever you are.

**Set up automation**

Configure cron jobs, webhooks, and file watchers. Your agent can run tasks on a schedule or when things change.

## Key concepts

| Concept | What it is |
|---|---|
| CLI | The command line interface. Run commands and conversations in your terminal. |
| Service API | An HTTP server that lets apps talk to your agent. |
| Connected Apps | Messaging apps that can reach your agent, such as Telegram or Slack. |
| Agents | The AI that processes your messages and uses tools. |
| Skills | Reusable instructions that teach the agent how to do things. |
| Memory | The agent can remember past conversations and context. |
