# Command reference

All CLI commands for OR3 Intern.

| Command | What it does |
|---|---|
| `chat` | Start an interactive chat session |
| `setup` | Run the interactive setup wizard |
| `settings` | View or change settings |
| `configure` | Edit configuration interactively |
| `init` | Create a default config file |
| `config-path` | Print the config file path |
| `version` | Print the version number |
| `serve` | Run all channels and automation |
| `service` | Run the HTTP API server |
| `agent` | Run a one-shot agent request |
| `doctor` | Run a diagnostic check |
| `embeddings` | Manage vector embeddings |
| `capabilities` | List available capabilities |
| `secrets` | Manage encrypted secrets |
| `audit` | View the audit log |
| `skills` | List and manage skills |
| `approvals` | View pending approvals |
| `devices` | Manage paired devices |
| `pairing` | Start device pairing |
| `scope` | View or change the active scope |
| `migrate-jsonl` | Import from JSONL format |
| `migrate-openclaw` | Migrate from OpenClaw |
| `connect-device` | Connect to a paired device |
| `status` | Show current status |
| `help` | Show this help message |
